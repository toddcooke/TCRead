# TCRead

iOS and macOS app to send free ebooks to your kindle.

### Development

Deploying iOS:

```
fastlane beta
fastlane release
```

Deploying macOS:

```
xcode -> archive -> distribute app
```

Take screenshots:

1. `fastlane snapshot`
1. Open ./fastlane/screenshots and upload to app store connect
